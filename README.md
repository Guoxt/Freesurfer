# Freesurfer

### 环境变量设置
```
nano /etc/profile
export FREESURFER_HOME=/usr/local/freesurfer  #这里FREESURFER_HOME的值是解压后freesurfer的地址，根据自己的情况填写
source /etc/profile

nano /etc/bash.bashrc
export FREESURFER_HOME=/usr/local/freesurfer  #这里FREESURFER_HOME的值是解压后freesurfer的地址，根据自己的情况填写
source ~/.bashrc

source /home/liyin/DATA_LL/freesurfer/SetUpFreeSurfer.sh
```
